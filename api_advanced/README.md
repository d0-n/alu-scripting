# API Advanced

Python scripts that query the Reddit API to practice working with APIs, pagination, and recursive functions.

## Requirements

- Ubuntu 14.04 LTS
- Python 3.4.3
- PEP 8 style
- Requests module

## Files

| File | Description |
|------|-------------|
| `0-subs.py` | Returns the number of subscribers for a given subreddit |
| `1-top_ten.py` | Prints the titles of the first 10 hot posts for a given subreddit |
| `2-recurse.py` | Recursively returns a list of all hot article titles for a given subreddit |
| `3-count.py` | Recursively counts and prints sorted keyword occurrences in hot article titles |

## How to use

```bash
python3 0-subs.py programming
python3 1-top_ten.py programming
python3 2-recurse.py programming
python3 3-count.py programming 'python java javascript'
```
